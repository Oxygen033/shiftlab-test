# Использованные технологии
- React, TypeScript
- Axios
- Zustand
- SWR
- react-form-hook

## Зависимости
NodeJS
yarn
docker-compose для запуска через Docker

## Запуск
С использованием Docker-compose
```bash
docker-compose up
```

Вручную
```bash
yarn install
yarn run dev
```
