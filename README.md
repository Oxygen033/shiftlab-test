# Тестовое задание шифтлаб 2024

## Требования
Необходимо создать форму авторизации с вводом номера телефона и отп кода.

## Требования к валидации


## Зависимости
NodeJS
yarn
docker-compose для запуска через Docker

## Запуск
С использованием Docker-compose
```bash
docker-compose up
```

Вручную
```bash
yarn install
yarn run dev
```
